SECURITY MODEL EXPLANATION :

🔐 Security Model

The security model is based on a Multi-Layer Defense Strategy:

Layer 1: Biometric Authentication

Triple verification system prevents spoofing attacks.

Layer 2: Identity-Link Verification

Phone number mapped to verified identity source.

Layer 3: Trust Score System

Each number and activity assigned dynamic trust value.

Layer 4: Fraud Detection AI

Detects:

Spoofed numbers

Abnormal behavior

Suspicious link patterns

Layer 5: Controlled Privacy Access

Even trusted users receive restricted access permissions.

Layer 6: Emergency Intelligence System

AI detects fall, inactivity, abnormal motion and triggers alert mechanism
